# Archlinux

#### 介绍
    所有脚本均在 virtualbox + archlinux 中测试

    archlinux基本安装步骤：
    分区，格式化，挂载，安装基本系统/软件，网络配置，chroot,passwd,重启！


#### 安装教程
    注：适用于系统安装后，第一次安装图形界面GUI
    注：中断操作(硬重启)后再次执行，会有问题，暂未解决
